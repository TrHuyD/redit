## About The Project

Redit is a full-stack Reddit clone.
<p align="right">(<a href="#readme-top">back to top</a>)</p>

### Features

* Community (subreddit) creation and subscription
* Post creation with text and image support
* Nested comment threads
* Upvote / downvote on posts and comments
* User authentication via NextAuth( Google OATH 2.0)
* Infinite scrolling feed
